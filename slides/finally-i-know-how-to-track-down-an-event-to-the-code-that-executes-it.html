Finally, I know how to track down an event to the code that executes it.

// Bind up a couple of event handlers
​$("#foo").on({
    click: function(){ alert("Hello") },
    mouseout: function(){ alert("World") }
});​​​

// Lookup events for this particular Element
​$._data( $("#foo")[0], "events" );
// > Object {click: Array[1], mouseout: Array[1]}
